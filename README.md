## Overview

This portfolio website provides an overview of my professional experience and projects. It is designed to be responsive and accessible, providing visitors with an easy way to learn about my work and skills.

## Technologies Used

- **HTML5**: Structure of the website.
- **CSS3**: Styling and layout.
- **JavaScript**: Interactivity and functionality.
- **Three.js**: WebGL-based background effects.
- **Showdown.js**: Markdown to HTML conversion.
- **Node.js**: Backend setup and build tools.
- **Webpack**: Module bundler.
- **NPM**: Package manager.

## Installation

To run this project locally, follow these steps:

1. **Clone the repository**:
    ```sh
    git clone https://github.com/yourusername/portfolio-website.git
    ```
2. **Navigate to the project directory**:
    ```sh
    cd portfolio-website
    ```
3. **Install dependencies**:
    ```sh
    npm install
    ```

## Usage

1. **Build the project**:
    ```sh
    npm run build
    ```
2. **Start the local server**:
    ```sh
    npm start
    ```

Your portfolio website should now be running on `http://localhost:8080`.

## Features

- **Responsive Design**: The website is fully responsive and works on all devices.
- **Interactive Background**: A dynamic WebGL background created using Three.js.
- **Markdown Support**: Content can be written in Markdown and converted to HTML using Showdown.js.
- **Project Showcase**: Displays my professional projects with detailed descriptions.
- **Contact Information**: Easy access to my contact details for potential employers or collaborators.
